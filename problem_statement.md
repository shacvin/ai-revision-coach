# Problem Statement: AI Revision Coach for Micro-Learning

Seekho users consume short educational videos quickly, but many forget key concepts soon after watching. Your challenge is to build an AI system that improves learning retention and repeat engagement after each video.

## Build

An AI-powered "Revision Coach" that, after a learner watches a video, can:

1. Generate a 60-second personalized recap (max 3 bullets).
2. Create adaptive quiz questions (difficulty based on learner performance).
3. Recommend the next best video based on weak concepts.
4. Trigger a next-day recall check (1 quick revision question).

## Important

1. AI must be the core intelligence, not a cosmetic chatbot wrapper.
2. Your system should show how learner behavior changes based on AI adaptation.

## Input (for prototype)

1. Video transcript + title/topic metadata.
2. Basic learner interaction data (quiz attempts, watch completion, topic history).

## Expected Output

1. Working prototype/demo of the revision flow.
2. Clear AI architecture (retrieval, personalization logic, model usage).
3. Learning impact metrics (example: quiz improvement, return rate, completion rate).

## Evaluation Focus

1. AI depth and originality
2. Learning impact
3. Practical execution
4. Scalability potential
